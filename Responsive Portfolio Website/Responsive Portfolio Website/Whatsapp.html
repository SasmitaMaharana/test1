This is a social media software for TIET. The software work on multiple platforms including Web App and Android. This software cater to all the basic needs of students, Alumni, Administration and faculty. This provide privilege to administrative and faculty members by tagging system. In this system, the users is able to upload their posts, connect with friends and also get access to centralized notice board. I also implemented an authentication system to provide security to this software.I also implemented map to help 1st year students navigate with ease.
